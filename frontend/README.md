# AI Resume–Job Match Engine

## Overview
An AI-powered system that evaluates how well a resume matches a job description using NLP techniques.

## Features
- TF-IDF vectorization
- Cosine similarity scoring
- Missing keyword detection
- Improvement suggestions

## Tech Stack
- FastAPI
- Scikit-learn
- React
- Axios

## How It Works
The system converts resume and job description into TF-IDF vectors and calculates cosine similarity to generate a match score.
